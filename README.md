## Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "Portfolio launch"
git remote add origin https://github.com/jagannath12333/portfolio.git
git branch -M main
git push -u origin main
```

Then go to: Repository Settings → Pages → Source: main / root → Save

Site goes live at: https://jagannath12333.github.io/portfolio
